# Task Manager - MERN Stack

A full-stack Task Manager application built with MongoDB, Express, React (Vite), and Node.js.

## Features

- ✅ Add new tasks
- ✅ View all tasks
- ✅ Delete tasks
- ✅ Input validation
- ✅ Loading and error states
- ✅ Responsive UI with Tailwind CSS v4.1

## Tech Stack

**Frontend:**
- React 18
- Vite
- Tailwind CSS v4.1
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- CORS

## Project Structure
```
task-manager/
├── backend/     # Express API server
└── frontend/    # React application
```
